# Web Framework Project

**OPTIONAL** install parcel html/ts server

```bash
npm i -g parcel
```

install JSON server (development)

```bash
npm i -g json-server
```

start JSON server

```bash
// in second terminal window
json-server -w db.json
```

To install the dependencies

```bash
yarn
```

To run the file

```bash
// ~./
parcel index.html
```
